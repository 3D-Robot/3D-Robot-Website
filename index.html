3D Robot

3D Robot is a robot that can move things around in the xyzrxryrzc field. It can also have toolheads that enable it to do certain tasks. It can also do assembly and finally it will be able to completely self replicate. A rendered model is in the picture to the left.

How it works


The Robot is made of two arms each has xyz linear movement and on it's end it has a gimbal which allows for rotation about the xyz called the rx, ry, rz axes. On the end of the gimbal there is a connector that will connect with computer manufactured parts The machine closely interacts with 3D Builders, which allow for many different types of computer aided manufacturing.
